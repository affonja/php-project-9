## About

Приложение на базе фреймворка Slim. Позволяет проверить доступность сайта, некоторые SEO-параметры, а также хранить
результаты проверок.

## Requirements

PHP 8.0, Composer, Make, Postgresql

## Install

> $ git clone git@github.com:affonja/php-project-9.git
>
> $ cd php-project-9
>
> $ make install
>
> $ make start
